So sicherst du deinen Lernfortschritt auf dem iPad

Deine Ergebnisse bleiben normalerweise nur in diesem Browser gespeichert. Mit Export kannst du eine Sicherungskopie erstellen und später wieder einlesen.

Export = Fortschritt sichern Tippe beim passenden Paket auf Export. Das iPad speichert eine kleine Datei meist im Ordner Downloads der App „Dateien“.
Import = Fortschritt zurückholen Tippe beim gleichen Paket auf Import. Wähle anschließend deine gespeicherte Datei in der App „Dateien“ aus.

Wichtig: Jedes Paket hat eine eigene Datei. Eine M-Datei gehört zum Mindeststandard, eine R-Datei zum Regelstandard und eine E-Datei zum Expertenstandard.
